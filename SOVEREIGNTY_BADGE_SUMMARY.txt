DEPLOYMENT MODE DETECTION:

  The useDeploymentMode() hook:
    - Calls System.keys() to fetch system settings
    - Checks LLMProvider field
    - Detection logic:
      * If provider includes: "ollama", "lmstudio", or "localai" → "local"
      * Otherwise defaults to "cloud-us"
    - Can be extended to detect EU cloud via API endpoint analysis

USAGE EXAMPLES:

  // Manual mode specification
  <SovereigntyBadge mode="local" size="md" showTooltip={true} />

  // Auto-detection with hook
  const [mode, setMode] = useState("cloud-us");
  useEffect(() => {
    async function detect() {
      const detected = await useDeploymentMode();
      setMode(detected);
    }
    detect();
  }, []);
  <SovereigntyBadge mode={mode} size="sm" />

  // Convenience auto-detect component
  <AutoDetectSovereigntyBadge size="md" className="custom-class" />
